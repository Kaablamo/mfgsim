MfgSim
Manufacturing Process Simulation Software

This package contains the Windows build of MfgSim.

What MfgSim does
- Build a process-flow model of a manufacturing system
- Estimate throughput
- Analyze resource utilization
- Identify queueing, blocking, and bottlenecks before hardware investment

Getting started
1. Extract the entire zip to a normal local folder.
2. Double-click MfgSim.exe.
3. MfgSim starts a local server on your machine and opens the application in your browser.

Important behavior
- The browser is the user interface.
- Closing the browser does not automatically stop the local MfgSim server.
- To fully close the packaged app, use the shutdown option in the Settings menu inside MfgSim.
- If needed, you can also terminate MfgSim from Task Manager.

Saving your work
- Models are saved as .mfgsim files.
- Use the Save button in the toolbar to write the current model to disk.
- Node and edge edits are applied to the current in-browser model automatically as you work.

Notes
- On some Windows systems, an unsigned executable may trigger a SmartScreen or security warning on first launch.
- Enterprise-managed PCs may block unsigned applications even if the software itself is valid.
